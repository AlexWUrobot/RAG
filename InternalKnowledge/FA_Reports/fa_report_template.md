# Failure Analysis Report Template

## Observation

Describe the sensor metrics, logs, and user-visible symptom.

## Specification Conflict

List the exact datasheet limit, interface rule, or internal engineering rule that conflicts with the observation.

## Hypothesis

State the most likely explanation, and note whether it is backed by internal FA evidence, datasheet evidence, or both.

## Recommended Action

List conservative next steps that an engineer can perform to validate or narrow the hypothesis.
